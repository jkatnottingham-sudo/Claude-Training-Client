# Claude API Training Exercises

This repo is used for training exercises used as part of Anthropic course - Building with the Claude API.

## Python-based Exercises

This repository contains practical exercises covering:

- **Invoking Claude API** - Basic API calls and interactions
- **Streaming** - Handling streaming responses from Claude
- **Prompt Evaluation** - Evaluating and testing prompt effectiveness
- **Prompt Engineering** - Crafting effective prompts for various use cases
- **Invoking Tools** - Implementing and using custom tools with Claude
- **Claude Tools** - Working with built-in tools including:
  - Web search
  - Text editor
  - Bash

## Getting Started

Ensure you have your Anthropic API key configured before running the exercises.
